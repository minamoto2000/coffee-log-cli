# Coffee Log CLI

コーヒー抽出のログを記録・閲覧できるPython製のCLIアプリです。  
抽出内容をJSONファイルに保存し、一覧表示・詳細表示・追加記録できます。

## アプリ概要

このアプリは、自宅でのコーヒー抽出内容を記録し、あとから振り返れるようにするために作成したPython製のCLIアプリです。  
抽出ログをJSONファイルに保存し、「一覧表示」「詳細表示」「追加記録」を通して管理できます。  
Python学習のアウトプットとして、データ構造、ファイル保存、関数分割、入力バリデーションの基礎を実践することを目的に開発しました。

## 作成背景

Pythonの学習を進めていく中で、インプットだけでは処理の流れやデータの扱いを十分に理解できず、知識が曖昧なままになりがちでした。  
そこで、ファイル保存・関数分割・入力バリデーションなどの基礎を実践するためのアウトプットとしてWebアプリを構想し、その足掛かりとして基礎的な技術で最低限の機能を持つCLIアプリを作成しました。  
また、日常の関心があるコーヒー記録を題材にすることでモチベーションを保ち、実際に使うことを意識して開発しました。

## できること

- 抽出ログの一覧表示
- 抽出ログの詳細表示
- 抽出ログの追加
- JSONファイルへの保存
- 入力値のバリデーション
  - 空文字チェック
  - 数値チェック
  - 範囲チェック
  
## 使用技術

- Python

## 実行環境

- Python 3.11 で動作確認
- 外部ライブラリ不使用（標準ライブラリのみ）

## 実行方法

リポジトリをクローンします。

```bash
git clone https://github.com/minamoto2000/coffee-log-cli.git
cd coffee-log-cli
```

アプリを起動します。

```bash
python main.py
```

## メニュー

アプリ起動後、以下のメニューを選択できます。

```text
1: ログ一覧
2: ログ詳細
3: ログを追加
4: 終了
```

## ファイル構成

```text
coffee-log-cli/
├── main.py      # メニュー表示、入力受付、各機能の呼び出し
└── logs.json    # 抽出ログ保存用ファイル
```

## 保存データ

抽出ログとして最低限振り返りに必要な情報に絞り、豆の名前・湯量・豆の量・スコアを保存する構成にしました。

- 豆の名前
- 湯量（g）
- 豆の量（g）
- スコア（1〜5点）

## JSON例

```json
{
  "bean_name": "Ethiopia",
  "water_g": 250,
  "beans_g": 15,
  "score": 4
}
```
