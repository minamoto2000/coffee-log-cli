# Coffee Log CLI

コーヒー抽出のログを記録・閲覧できるPython製のCLIアプリです。  
抽出内容をJSONファイルに保存し、一覧表示・詳細表示・追加保存ができます。

## アプリ概要

このアプリは、自宅でのコーヒー抽出内容を記録し、あとから振り返れるようにするために作成したCLIアプリです。  
抽出データをJSONファイルに保存し、一覧表示や詳細表示を通して記録を確認できます。

## 作成背景

Pythonの学習を進めていく中で、インプットだけでは処理の流れやデータの扱いを十分に理解できず、知識が曖昧なままになりがちでした。  
そこで、ファイル保存・関数分割・入力バリデーションなどの基礎を実践するためのアウトプットとして、基礎的な技術で最低限の機能を持つCLIアプリを作成しました。  
また、日常の関心があるコーヒー記録を題材にすることでモチベーションを保ち、実際に使うことを意識して開発しました。

## できること

- 抽出ログの一覧表示
- 抽出ログの詳細表示
- 抽出ログの追加
- JSONファイルへの保存
- 入力値のバリデーション
  - 空文字チェック
  - 数値チェック
  - 範囲チェック

## 使用技術

- Python
- JSON
- Git / GitHub

## 実行方法

リポジトリをクローンします。

```bash
git clone https://github.com/minamoto2000/coffee-log-cli.git
cd coffee-log-cli
```

アプリを起動します。

```bash
python main.py
```

## メニュー

アプリ起動後、以下のメニューを選択できます。

```text
1: ログ一覧
2: ログ詳細
3: ログを追加
4: 終了
```

## 保存データ

1件の抽出ログとして、以下の情報を保存します。

- 豆の名前
- 湯量（g）
- 豆の量（g）
- スコア（1〜5点）
- 工夫した点

## 工夫した点

- 関数ごとに役割を分け、処理の見通しをよくした
- JSONファイルが存在しない場合や壊れている場合でも、アプリが動作するようにした
- ユーザー入力に対して、空文字・数値・範囲のチェックを実装した
- 一覧表示と詳細表示の役割を分け、情報を見やすく整理した
